# Pet project

### Описание
Проект представляет собой распределённую систему для обработки данных о книгах, где
REST API продюсер принимает через ручку /books информацию о книге и отправляет её в
Kafka топик, откуда консумер асинхронно забирает сообщения, сохраняет книги в
PostgreSQL и одновременно предоставляет gRPC API сервиса аналитики, позволяющий
запросить общее количество добавленных книг; вся инфраструктура (Kafka, БД) и
микросервисы оркестрируются через Docker Compose.
Написаны unit-тесты и интеграционные тесты, используя testcontainers.

### Сборка
Перед сборкой необходимо создать `.env` в корне репо и в [consumer](./consumer/), основываясь на `.env.example`
```
docker-compose up
```
